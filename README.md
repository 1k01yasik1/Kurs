# Визуализация орбитального движения спутника

Приложение демонстрирует компьютерно-графическую сцену с процедурно
сгенерированной планетой, освещением от звезды и спутником, движущимся по
орбите. Рендеринг выполняется программным трассировщиком лучей, который
подсчитывает тени от спутника на поверхности планеты и затенение самого
спутника планетой. Интерфейс напоминает классические лабораторные работы по
компьютерной графике и позволяет управлять параметрами сцены.

## Возможности

- Процедурная генерация высот планеты на основе seed, заданного пользователем.
- Вращение камеры вокруг сцены, перенос центра и масштабирование.
- Настройка радиуса орбиты и угловой скорости спутника.
- Трассировка лучей с учётом диффузного освещения и зеркальных бликов.
- Расчёт теней от спутника на планету и затенение спутника планетой.
- Запись последовательности кадров в директорию `frames/`.

## Установка

```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\\Scripts\\activate
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

После запуска появится окно с панелью управления слева и окном визуализации
справа. Используйте элементы управления, чтобы изменять положение камеры,
параметры орбиты и seed генерации планеты. Кнопка «Начать запись» сохранит
последовательность текущих кадров в папку `frames/`.

## Структура данных планеты

Поверхность планеты представлена матрицей высот в полярных координатах
(широта × долгота). Для каждой точки вычисляется суммарная высота, полученная из
фрактального шума и поля кратеров. Значения конвертируются в радиус в процессе
трассировки лучей, что позволяет получить детализированную поверхность без
хранения треугольной сетки.
