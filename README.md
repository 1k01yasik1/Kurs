# Kurs

Программа визуализации движения спутника вокруг протопланетного тела на Python.

## Запуск

1. Установите зависимости (желательно в виртуальном окружении):

   ```bash
   pip install -r requirements.txt
   ```

2. Запустите приложение:

   ```bash
   python main.py
   ```

## Управление

- `Space` — пауза/продолжение анимации.
- `+` / `-` — ускорение или замедление моделируемого времени.
- Стрелки `←` `→` — изменение орбитального радиуса, `↑` `↓` — изменение скорости запуска.
- `[` `]` — изменение гравитационного параметра центрального тела.
- `T` — отображение орбитального следа, `C` — очистка следа.
- `G` — включение/выключение координатной сетки.
- `Z` / `X` — уменьшение/увеличение масштаба сцены.
- `R` — сброс параметров к настройкам по умолчанию, `N` — случайная орбита.
- `Esc` — выход из программы.

## Возможности

- Моделирование движения спутника на основе численного интегрирования уравнений движения.
- Отображение траектории спутника, основных орбитальных параметров и характеристик движения.
- Настройка гравитационного параметра, стартовой скорости, орбитального радиуса и масштаба визуализации.
